****************
Dummy Microphone
****************

This is a simple dummy driver for simulating a microphone (mic)
<https://en.wikipedia.org/wiki/Microphone>`__.

Usage
=====

Usage Reference
---------------

::

   usage: dummy-mic [-c CHANNELS] [-s KHZ] [-b KBITS] FILE

Usage Example
-------------

::

   usage: dummy-mic -c2 -s48 -b64 hello.wav
